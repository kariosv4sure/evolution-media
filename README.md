# Evolution Media

Technology-driven platform for JAMB/WAEC success, digital empowerment, and advertising services.

## Features
- User authentication (login/register)
- AI chat assistant (Groq API)
- Admin dashboard with user management
- JAMB & WAEC resource pages
- Contact form with WhatsApp integration
- Responsive dark theme with pulse red accents

## Tech Stack
- Flask (Python)
- PostgreSQL (Render)
- SQLAlchemy (ORM)
- Flask-Login (auth)
- Groq API (AI)
- HTML/CSS/JS

## Deployment
Deployed on Render.com

## Admin Credentials
- Username: evolution_admin
- Password: set in .env

## Local Setup
1. Clone repo
2. Create virtual environment
3. `pip install -r requirements.txt`
4. Set up .env file
5. `python app.py`
